INFORMAIONS----------------------------
Vous trouverez les différents packages dans le dossier "src".

Le dossier src contient les packages:
-representation : modélisation d'un problème en logique variable-valeur
-solver: résolution de problèmes
-planning: recherche de la suite d'action à effectuer pour obtenir la résolution du problème
-datamining: extraction des connaissances
-example: exemple d'application des différents algorithmes, dans notre cas une construction de maisons.
